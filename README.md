### Installation

**1. Clone the repository**

```bash
git clone https://github.com/O2ymandias/FreshCart.AdminDashboard-Frontend.git
cd FreshCart.AdminDashboard-Frontend
```

**2. Install dependencies**

```bash
npm install
```

**3. Start the development server**

```bash
ng serve
```

Navigate to `http://localhost:4200`
